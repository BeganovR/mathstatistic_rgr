# mathstatistic_rgr

Расчётно-графическая работа по математической статистике.  
Генерирует Excel-файл **`mathstat_rgr.xlsx`** с полной статистической обработкой выборки X (N = 121..240, 120 значений).

## Содержимое Excel-файла

| Лист | Описание |
|------|----------|
| 1. Исходные данные | Столбцы N (121–240) и X |
| 2. Вариационный ряд | Выборка X, отсортированная по возрастанию |
| 3. Стат. ряд | Статистический ряд: xᵢ, kᵢ, wᵢ; R, h |
| 4. Интервальный ряд | Интервалы, kᵢ, wᵢ, плотность wᵢ/h; Мода Mo, Медиана Me |
| 5. Полигон и F(x) | Полигон частот, эмпирическая ФР F*(x), гистограмма — с графиками |
| 6. Начальные моменты | α₁, α₂, α₃, α₄ |
| 7. Характеристики выборки | x̄в, Dв, σв, β₂, β₃, β₄, асимметрия A, эксцесс E |
| 8. Точечные оценки | Методы статистических оценок и моментов |
| 9. Интервальные оценки | Доверительные интервалы для x̄ и σ при γ = 0.95 |

## Быстрый старт

Готовый файл `mathstat_rgr.xlsx` уже включён в репозиторий — скачайте и откройте его напрямую.

## Запуск скрипта самостоятельно

### Требования

- Python 3.8 или выше

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Генерация Excel-файла

```bash
python generate_excel.py
```

После выполнения в текущей директории появится файл `mathstat_rgr.xlsx`.
