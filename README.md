# mathstatistic_rgr

Расчётно-графическая работа по математической статистике.  
Генерирует Excel-файл **`mathstat_rgr.xlsx`** с полной статистической обработкой выборки X (N = 121..240, 120 значений).

## Содержимое Excel-файла (структура по ТЗ)

| Лист | Описание |
|------|----------|
| 1. Выборка X | Столбцы N (121–240) и X (исходные данные) |
| 2. Вариационный ряд | Выборка X, отсортированная по возрастанию |
| Табл.1 - Стат. ряд | Статистический ряд: xᵢ, kᵢ, wᵢ = kᵢ/n; Σ; R, h |
| Табл.2 - Интервальный ряд | Интервалы, середина xᵢ*, kᵢ, wᵢ, wᵢ/h; Мода Mo, Медиана Me; **Гистограмма** |
| Табл.3 - Полигон и F(x) | Точечный ряд; F*(x); **Полигон частот** и **График F*(x)** |
| Табл.4 - Начальные моменты | xᵢ·wᵢ, xᵢ²·wᵢ, …; α₁–α₄; x̄в, Dв, σв, A, E |
| 7. Точечные оценки | Метод стат. оценок (S², S) и метод моментов |
| 8. Интервальные оценки | Доверительные интервалы для x̄ и σ при γ = 0.95 |

## Быстрый старт

Готовый файл `mathstat_rgr.xlsx` уже включён в репозиторий — скачайте и откройте его напрямую.

## Запуск скрипта самостоятельно

### Требования

- Python 3.8 или выше

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Генерация Excel-файла

```bash
python generate_excel.py
```

После выполнения в текущей директории появится файл `mathstat_rgr.xlsx`.
